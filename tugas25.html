Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel kita gunakan dengan header dan footer.Karena tag article memiliki type block, artinya elemen inimeiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama.Misal kita memiliki halaman blog halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelakan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menemukan lebarnya
